# plot-corona
Hobby project to explore Wikidata and graph plotting by example of the corona virus outbreak 2020.
The available data on Wikidata is surprisingly sparse.

# usage
You will need to add a `user-config.py` for the pywikibot. Example content:
```
mylang = 'wikidata'
family = 'wikidata'
usernames['wikidata']['wikidata'] = u'MyUserNameWhichDoesNotExist'
```
